Side Scrolling Game: Billy the Baby Blue Whale

Slide Scrolling Game!! I partnered with Lily on this one. The object of the game is to help Billy the Baby Blue Whale catch up to his mother while avoiding the trash and predators. The player uses the up and down arrow keys to navigate through the water. We used Scratch to create this project.


This is the code we used for Billy. There were more blocks with the same code, but for different variables. Essentially, this code says that when the green flag is clicked, Billy will give the "instructions" and the score will be reset to zero. If he touches an obstacle, in case the shark, he will say "I've been eaten!" and the score will show. Then, the player will be taken to the "high score!" page if they beat their high score, and then the "game over" page.


This is the code for the sand. It uses cloning to repeatedly move across the screen to make it look like the back drop is moving.


This is the code for the obstacles that come at Billy. They are coded to create clones of themselves that move across the screen, and then delete the previous clone so just one remains. This creates the illusion that the shark is moving across the screen.
